FORMAT TEXT

You can also format text directly by using the other controls on the home tab. Most controls offer a choice of using the look from the current theme or using a format that you specify directly. To change the overall look of your document, choose new Theme element on the Page Layout tab.

To change the looks available in the Quick Style gallerry, use the Change Current Quick Style Set command. Both the Themes gallery and the Quick Styles gallery provide reset commnands so that you can always restore the look of your documnet to the original contained in your current template.
